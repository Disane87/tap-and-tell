# Customization
Every owner can customize their tenant's guestbook appearance and settings based on their subscription plan including 
background images (upload), colors, font, texts etc.

## Features
- [ ] Basic customization for Free plan (name only)
- [ ] Full customization for paid plans (colors, logo, welcome message, background image)
- [ ] Questions customization (enable/disable, custom text)
- [ ] Preview mode for customization changes
- [ ] Save and publish changes
- [ ] Reset to default option
