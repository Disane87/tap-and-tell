# Export to PDF

Download guestbook entries as a keepsake document.

## Features

- Generate PDF with all entries
- Include photos, names, messages, dates
- Customizable layout (grid or list)
- Cover page with event name/date
- Download button on guestbook page

## Implementation

- `jspdf` or `@react-pdf/renderer` equivalent for Vue
- Server-side PDF generation endpoint
- `/api/export/pdf` route
- Styled templates for entries
