# QR Code Fallback

Generate QR code for devices without NFC.

## Features

- Generate QR code linking to guestbook URL
- Include event parameters in URL
- Downloadable QR code image
- Printable format
- Admin page to generate/customize

## Implementation

- `qrcode` library
- `/admin/qr` page for generation
- Customizable URL parameters
- PNG/SVG export options
